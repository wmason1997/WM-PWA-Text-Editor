## Installation <a name="installation-instructions"></a>
Clone this repository locally. Then in terminal, run:
```md
npm i
```

## Usage <a name="usage-instructions"></a>
In terminal, run:
```md
npm run start
```

and open in browser at the specified port. In this case, it is 3000.
